Token Types

输入端 token 分为 fresh、cache read、cache write；输出端 token 单独统计。

Metric Definition Formula Provider Fields
{{ token_input("Fresh") }} Mutually exclusive new-input component used by UI tokenbar and Total Tokens. Anthropic-like: input_tokens; OpenAI/Codex: Provider Request Input - Cache Read Provider Request Input remains raw accounting metadata; it is not stacked as Fresh beside cached input.
{{ token_cache_read("Cache Read") }} Provider/broker accounting for cache-hit input tokens. cache_read_input_tokens Anthropic: cache_read_input_tokens; OpenAI: cached_tokens; Codex: cached_input_tokens; Qoder: cache_read_input_tokens / cached_tokens when reported.
{{ token_cache_write("Cache Write") }} Input tokens written into provider prompt cache for later reuse. cache_write_input_tokens Anthropic: cache_creation_input_tokens; OpenAI/Codex/Qoder: Not reported
{{ token_output("Output") }} Visible model output tokens, excluding hidden reasoning where reported separately. output_tokens Anthropic: output_tokens; OpenAI: completion_tokens - reasoning_tokens; Codex/Qoder: provider total output field when available.
Input-side Component Total All input-side token buckets combined. Fresh + Cache Read + Cache Write Derived from available provider fields; missing buckets are not treated as zero unless the provider explicitly reports zero.
Total Tokens Input-side total plus visible output. Fresh + Cache Read + Cache Write + Output Used by Dashboard, Sessions, Project Detail, and Session Detail token cells.

Derived Metrics

派生指标只解释聚合口径，不替代业务页面常驻指标。

Metric Formula Interpretation
Cache Read Ratio Cache Read / Input-side Component Total cache reuse health; used in Dashboard token charts, Sessions token cell, and Session Detail KPIs.
Cache Write Ratio Cache Write / Input-side Component Total Shows active prompt cache creation in Dashboard and Session Detail token diagnostics.
Output/Input Ratio Output / Input-side Component Total Highlights output density in Dashboard charts and Session Detail payload analysis.
Tools / Round tool_calls / rounds Used in Sessions and Session Detail to compare interaction density.
Tokens / Round Total Tokens / rounds Used in Session Detail diagnostics to find oversized rounds.
Tokens / Minute Total Tokens / (duration_minutes) Used in Dashboard throughput charts and Sessions duration comparisons.
Failed / Session failed_tool_count / sessions Used by Dashboard Failure Signals and Sessions failure sorting.

Provider Field Mapping

字段缺失时使用 Not reported；由其它字段推导时标记 Estimated。

Provider Model Family Fresh Cache Read Cache Write Output
Anthropic Claude 4 input_tokens cache_read_input_tokens cache_creation_input_tokens output_tokens
qwen Anthropic compatible qwen* input_tokens cache_read_input_tokens cache_creation_input_tokens output_tokens
OpenAI gpt-* / o-series input_tokens cached_tokens Not reported completion_tokens - reasoning_tokens
Codex gpt-* codex input_tokens cached_input_tokens Not reported tokens_used aggregate
Qoder custom models input_tokens cache_read_input_tokens / cached_tokens cache_creation_input_tokens when reported output_tokens / completion_tokens

Round Signals

Round-level signals explain what Session Detail Trace and Dashboard failure summaries highlight.

Signal Definition Used In
round A grouped assistant interaction and its related model/tool activity. Session Detail Trace / Sessions Rounds column
step A model call, tool call, or displayed trace item inside a round. Session Detail Trace
batch A set of tool calls grouped by timing or trace adjacency. Session Detail Trace / Tool Hotspots
tool Tool invocation count, duration, result, and failure status. Session Detail Trace / Dashboard Tool Hotspots
subagent Delegated agent activity attached to a parent session or tool result. Session Detail Trace / Sessions Subagents column
failure failed tool result, attribution error Session Detail Trace / Dashboard Failure Signals
