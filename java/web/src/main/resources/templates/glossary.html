Token Types

输入端 token 分为 Fresh、Cache Read、Cache Write；输出端 token 单独统计。

Metric Definition Formula Provider Fields
Fresh 互斥的新输入组件，用于 UI tokenbar 和 Total Tokens。 Anthropic-like: input_tokens; OpenAI/Codex: Provider Request Input - Cache Read Provider Request Input 只作为原始计量元数据，不与缓存输入并列叠加为 Fresh。
Cache Read Provider/broker 上报的缓存命中输入 token。 cache_read_input_tokens Anthropic: cache_read_input_tokens; OpenAI: cached_tokens; Codex: cached_input_tokens; Qoder 上报时使用 cache_read_input_tokens / cached_tokens。
Cache Write 写入 provider prompt cache、供后续复用的输入 token。 cache_write_input_tokens Anthropic: cache_creation_input_tokens; OpenAI/Codex/Qoder: Not reported
Output 模型可见输出 token；如 hidden reasoning 单独上报，则不计入可见输出。 output_tokens Anthropic: output_tokens; OpenAI: completion_tokens - reasoning_tokens; Codex/Qoder 有 provider 输出字段时优先使用。
Input-side Component Total 所有输入侧 token 组件的合计。 Fresh + Cache Read + Cache Write 由可用 provider 字段推导；字段缺失不等同于 0，除非 provider 明确上报 0。
Total Tokens 输入侧组件合计加可见输出。 Fresh + Cache Read + Cache Write + Output 用于 Dashboard、Sessions、Project Detail 和 Session Detail 的 token 单元格。

Derived Metrics

派生指标只解释聚合口径，不替代业务页面常驻指标。

Metric Formula Interpretation
Cache Read Ratio Cache Read / Input-side Component Total 衡量 cache reuse health；用于 Dashboard token 图表、Sessions token 单元格和 Session Detail KPI。
Cache Write Ratio Cache Write / Input-side Component Total 展示 prompt cache 写入活跃度，用于 Dashboard 和 Session Detail token 诊断。
Output/Input Ratio Output / Input-side Component Total 突出输出密度，用于 Dashboard 图表和 Session Detail payload 分析。
Tools / Round tool_calls / rounds 用于 Sessions 和 Session Detail 对比交互密度。
Tokens / Round Total Tokens / rounds 用于 Session Detail 诊断超大 round。
Tokens / Minute Total Tokens / (duration_minutes) 用于 Dashboard 吞吐图表和 Sessions 时长对比。
Failed / Session failed_tool_count / sessions 用于 Dashboard Failure Signals 和 Sessions failure 排序。

Provider Field Mapping

字段缺失时使用 Not reported；由其它字段推导时标记 Estimated。

qwen Anthropic compatible qwen* input_tokens cache_read_input_tokens cache_creation_input_tokens output_tokens

Round Signals

Round 级信号说明 Session Detail Trace 和 Dashboard failure 摘要会高亮什么。

Signal Definition Used In
round 一次分组后的 assistant 交互及其相关 model/tool 活动。 Session Detail Trace / Sessions Rounds column
step round 内的 model call、tool call 或可见 trace 项。 Session Detail Trace
batch 按时间或 trace 相邻关系分组的一组 tool calls。 Session Detail Trace / Tool Hotspots
tool 工具调用次数、耗时、结果和失败状态。 Session Detail Trace / Dashboard Tool Hotspots
subagent 挂在父 session 或 tool result 下的委派 agent 活动。 Session Detail Trace / Sessions Subagents column
failure failed tool result、attribution error Session Detail Trace / Dashboard Failure Signals
