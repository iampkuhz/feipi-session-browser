# UI Context v15

The target UI baseline is `docs/ui/hifi/session_browser_hifi_v15/`.

Current product UI should be upgraded to this baseline. Existing code structure and routes are implementation constraints, not visual baseline.

Important choices:

```text
light theme only by default
compact trace workbench
no right-side inspector
trace and calls are merged
Projects page shows recent project list only
Agents page remains separate
Context / Response / Result modals have different content contracts
```
