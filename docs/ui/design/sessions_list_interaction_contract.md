# Sessions 列表交互要求

## 搜索

- 输入关键词后实时更新或提交更新。
- 搜索状态必须反映在 URL 或页面状态中。
- 清空搜索后恢复当前其它过滤条件。

## 过滤

- 过滤控件变化后结果更新。
- Active Filters 展示所有已生效过滤项。
- 每个 Active Filter 可单独移除。
- 清空全部过滤后回到默认列表。

## 排序

- 可排序表头整体可点击。
- 当前排序列展示方向。
- 再次点击同一列切换方向。
- 排序不丢失搜索、过滤和分页上下文。

## 分页

- 切换页码保留搜索、过滤和排序状态。
- page size 改变后回到有效页。
- 首页禁用或隐藏上一页。
- 尾页禁用或隐藏下一页。

## 行交互

- 点击行进入 Session Detail。
- 行内按钮只执行自己的动作。
- 复制动作不触发行跳转。
- 键盘焦点必须能到达主要控件。
