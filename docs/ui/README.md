# UI 页面功能要求

`docs/ui/` 是页面功能要求的维护目录。产品代码、模板、CSS、JS、测试和质量门应对齐这里的要求。

## 入口

- `contracts/03-page-contracts.md`：页面范围和逐页功能要求。
- `contracts/01-global-ui-contract.md`：全局布局、密度、表格、token 和导航要求。
- `contracts/02-component-contracts.md`：共享组件要求。
- `contracts/05-button-icon-behavior.md`：按钮、图标和可点击元素行为要求。
- `contracts/06-validation-contract.md`：验证入口和质量要求。
- `contracts/07-data-contract.md`：页面数据、空态和不可用原因展示要求。
- `contracts/css-load-order.md`：CSS 加载顺序和 ownership 要求。
- `contracts/shell-state-contract.md`：Shell 状态要求。
- `design/sessions_list_component_system.md`：Sessions 列表组件要求。
- `design/sessions_list_interaction_contract.md`：Sessions 列表交互要求。
